# Drone-Delivery-System-WT
Technology Stack: 

MERN: MongoDB, Express, React, Node

Functional Requirements
1. Backend (Reuse + Improve)
   Includes authentication, APIs, and database.
2. Frontend Development
    Developed frontend with pages: Login, Register, Dashboard, Create Delivery, View Deliveries, Update Delivery, Drone Listing.
3. Integration
   Connected frontend with backend APIs, store JWT token, performed CRUD operations.
4. Required Features
   Form validation, error/success messages, loading indicators, protected routes, logout.
